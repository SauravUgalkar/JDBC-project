# JDBC-project

Database CRUD System (JDBC)
A data persistence application that connects Java to a MySQL database.

Key Features: Full CRUD (Create, Read, Update, Delete) capabilities for user records.

Tech Highlight: Uses JDBC (Java Database Connectivity) and PreparedStatement to handle SQL queries securely and prevent SQL injection.

Logic: Automatically detects if the database table exists and creates it if missing, ensuring seamless setup.

The Cart Management System is a Java-based console application designed to demonstrate Database Connectivity (JDBC). The primary objective of this project is to simulate a basic e-commerce backend where an administrator can manage products, and a user can manage a shopping cart.

Key Features:

Product Management: Add, Update, and Delete products from the store inventory.

Cart Management: Add products to a user cart, update quantities, remove items, and view the cart.

Data Persistence: Uses MySQL database to store product and cart information permanently.

Security: Utilizes PreparedStatement to prevent SQL injection and manage secure database queries.
